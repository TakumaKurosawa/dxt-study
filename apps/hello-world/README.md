# Hello World DXT Extension

シンプルな Hello World DXT 拡張機能です。

## 機能

- **say_hello**: 指定された名前に対して挨拶を返します
- **get_time**: 現在の時刻を返します

## 使用方法

Claude Desktop にインストール後、以下のように使用できます：

```
私の名前は太郎です。挨拶してください。
```

```
現在の時刻を教えてください
```

## ローカルテスト

```bash
cd server
node index.js
```

## パッケージング

```bash
dxt pack
```
